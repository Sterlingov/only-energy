Сайт с обзорами энергетиков

Бот ловит посты в телеграм-канале, добавляет в БД postgresql. RESTful API на Go отдает данные фронтенду

Для запуска бэкенда:
/backend/bot/.env
BOT_TOKEN = 
CHANNEL_ID =
POSTGRES_PASSWORD =

/backend/server/.env
POSTGRES_USER = 
POSTGRES_PASS = 
POSTGRES_DB = 
DOMAIN = 
SECRET_HEADER =
